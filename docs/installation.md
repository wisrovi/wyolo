# Installation

You can install `wyolo` directly from PyPI:

```bash
pip install wyolo
```

## From Source

If you want to install the latest development version:

```bash
git clone https://github.com/wisrovi/wyoloservice2_worker.git
cd wyoloservice2_worker
pip install -e .
```

## Dependencies

`wyolo` requires Python 3.8+ and several key libraries:
- `ultralytics`
- `mlflow`
- `dvc`
- `redis`
- `GPUtil`
